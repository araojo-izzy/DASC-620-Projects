# DASC-620-Projects
Assignments from ODU's DASC/CS 620: Intro to Data Science

## Activities
### bfast - List Filtering with Lambda Functions
A simple exercise using Python's built-in filter() function combined with 
a lambda expression to filter a list of breakfast items by word length.

### Activity 6 - NumPy Array and DataFrame Manipulation
Creates a random NumPy array, converts it to a pandas DataFrame, transposes 
it, and replaces all values less than 40 with zero.

### Activity 7 - Applying Lambda Functions to DataFrames
Creates a random NumPy array as a DataFrame, applies a lambda function to 
compute the square root of every value using map(), and calculates the row 
sums using apply().

## Tools Used
- Python 3
- numpy
- pandas
- math
